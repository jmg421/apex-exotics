## SPECIFIC HAZARD CONTROLS

**Drywall Dust:**
- [ ] Dust masks worn during cutting
- [ ] Ventilation maintained
- [ ] Sweep frequently (Weston's job)
- [ ] No eating/drinking in dusty area

**Sharp Edges:**
- [ ] Gloves worn when handling cut drywall
- [ ] Cut edges pointed away from body
- [ ] Scrap pieces handled carefully
- [ ] No running hands along edges

**Heavy Lifting:**
- [ ] Two-person lift for full sheets
- [ ] Bend knees, straight back
- [ ] No twisting while carrying
- [ ] Ask for help if too heavy

**Power Tool Specific:**
- [ ] Circular saw: Material supported, won't pinch blade
- [ ] Drill: Bit secure, material clamped/stable
- [ ] Oscillating tool: Blade tight, hands clear
- [ ] All tools: Unplug to change blades/bits

**Electrical:**
- [ ] GFCI outlet used if available
- [ ] Cords not in water or wet areas
- [ ] Extension cords rated for tool amperage
- [ ] No daisy-chaining extension cords

---

## INJURY RESPONSE PROTOCOL

**Minor Cuts/Scrapes:**
- [ ] Stop work immediately
- [ ] Clean wound with water
- [ ] Apply antiseptic
- [ ] Bandage appropriately
- [ ] Document injury (what happened, when, how)
- [ ] Monitor for signs of infection

**Serious Injury (Deep cut, eye injury, broken bone):**
- [ ] Stop all work immediately
- [ ] Call 911 if needed
- [ ] Apply first aid (pressure for bleeding, don't remove objects from eyes)
- [ ] Keep injured person calm and still
- [ ] Have someone meet ambulance if called

**Eye Injury:**
- [ ] Do NOT rub eyes
- [ ] Flush with water for 15 minutes
- [ ] Seek medical attention immediately
- [ ] Do NOT remove embedded objects

## SAFETY RULES - POST VISIBLY

**THE 5 NON-NEGOTIABLES:**

1. **EYES PROTECTED** - Safety glasses on during cutting/mounting
2. **REPORT INJURIES** - All injuries reported immediately, no exceptions
3. **STOP WORK AUTHORITY** - Anyone can call timeout for safety concerns
4. **ONE TOOL, ONE PERSON** - No sharing power tools during operation
5. **ASK IF UNSURE** - Better to ask than guess with safety
